#⚛️ Computação Quântica — Conceitos Fundamentais

**Acesse aqui:**
[Computação Quântica](https://baccarinha.github.io/computacaoquantica/)

🎯 **Objetivo do Projeto**

Desenvolver uma calculadora digital simples capaz de realizar operações matemáticas básicas, demonstrando o uso de lógica de programação e interatividade em aplicações web.

🧩 **Funcionalidades do Sistema**

|O sistema apresenta as seguintes funcionalidades:|
|--|
🧮 Realização de adição
➖ Realização de subtração
✖️ Realização de multiplicação
➗ Realização de divisão
🔢 Entrada de valores numéricos
📊 Exibição automática do resultado
🖥️ Interface simples e intuitiva
⚡ Resposta rápida aos comandos do usuário
🛠️ Tecnologias Utilizadas

|Este projeto foi desenvolvido utilizando:|
|--|
HTML5 — Estruturação da interface
CSS3 — Estilização visual
JavaScript — Lógica das operações matemáticas
GitHub Pages — Hospedagem do projeto
Inteligência Artificial (IA) — Apoio na criação das ideias e organização
